Adds next GIF frame. The frame is not written immediately, but is actually deferred until the next frame is received so that timing data can be inserted. Invoking finish() flushes all frames. If setSize was invoked, the size is used for all subsequent frames. Otherwise, the actual size of the image is used for each frames.

true if successful.

Content copied to clipboard

Adds next GIF frame to the specified position. The frame is not written immediately, but is actually deferred until the next frame is received so that timing data can be inserted. Invoking finish() flushes all frames. If setSize was invoked, the size is used for all subsequent frames. Otherwise, the actual size of the image is used for each frame. See page 11 of http://giflib.sourceforge.net/gif89.txt for the position of the frame

Return

true if successful.

Parameters

im

BufferedImage containing frame to write.

x

Column number, in pixels, of the left edge of the image, with respect to the left edge of the Logical Screen.

y

Row number, in pixels, of the top edge of the image with respect to the top edge of the Logical Screen.
